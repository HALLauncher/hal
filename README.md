# Hoi4 Alternative Loader (HAL)
My attem to create alternative loader for [Hearts of Iron IV](https://store.steampowered.com/app/394360/Hearts_of_Iron_IV/) and other paradox games

# Build and Run

* Init
```
yarn
```
* Build
```
yarn tauri build
```
* Run
```
yarn tauri dev
```
